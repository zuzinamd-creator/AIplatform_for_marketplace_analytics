# Визуализация прибыльности (seller-friendly)

Цель: визуализация должна быть “ежедневной”, без ощущения enterprise-дашборда.

## Основные графики

- **Выручка vs прибыль** (дневной тренд)
- **Логистика vs реклама vs штрафы** (что “съедает” прибыль)
- **Маржа (дневная)** (как меняется эффективность)
- **Возвраты (динамика)** (качество товара/контента/логистики)

## Принципы визуализации

- Четкие подписи и единицы (₽ / % / шт.)
- Минимум визуального шума: скрываем оси там, где они не помогают
- Цвета с понятной семантикой (выручка — холодный, прибыль — зеленый, штрафы — красный/розовый)
- Доверие к данным — отдельным блоком (полнота, предупреждения)
