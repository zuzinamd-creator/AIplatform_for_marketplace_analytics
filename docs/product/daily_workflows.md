# Ежедневные workflow продавца

## “Что делать сегодня”

1. `/app/today` — посмотреть “что требует внимания”.
2. Открыть рекомендацию → отметить **Сделано** / **Сделать сегодня** / **Жду данные**.
3. Добавить заметку и напоминание.

## “Разобрать убыточные SKU”

1. `/app/economics` — сортировка по маржинальному вкладу.
2. Провалиться в `/app/economics/sku/:sku` — тренды и объяснения.
3. Если данные неполные — сначала исправить “доверие к данным” (себестоимость/сверка выплат).

## “Проверить загрузки”

1. `/app/reports/upload` — загрузить отчёты.
2. Секция “Что нужно загрузить…” — подсказки по coverage и integrity.
