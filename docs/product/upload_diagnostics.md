# Диагностика загрузки отчётов

## Принцип

Ошибки загрузки должны быть **продавцу понятны**: что не так, как исправить, что загрузить для точности.

## Что сделано

- `validate_report_file()` возвращает seller‑friendly кодированные ошибки:
  - `[wb_excel_unrecognized] …` — не распознан формат WB Excel
  - `[invalid_values] …` — некорректные значения (числа/количества)
- UI `/app/reports/upload` показывает:
  - заголовок + понятный текст ошибки
  - блок “Что нужно загрузить для более точного анализа” (coverage‑driven)

## Что загрузить для точности

Блок подсказок основан на `GET /api/v1/analytics/coverage`:

- `recommendations[]` (например: загрузить себестоимость, заполнить пропуски периодов)
- `warnings[]` / `financial_completeness_score`
