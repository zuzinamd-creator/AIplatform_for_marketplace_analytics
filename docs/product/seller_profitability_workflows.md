# Сценарии продавца: прибыльность и склад

## Ежедневный сценарий: “Что съедает прибыль?”

1. Открыть `/app/economics`
2. Отсортировать по “Маржинальный вклад”
3. Найти убыточные/низкомаржинальные SKU
4. Провалиться в SKU (`/app/economics/sku/:sku`)
5. Посмотреть блок “Логистика vs реклама vs штрафы” и динамику возвратов
6. Если есть предупреждения по данным — сначала устранить их (себестоимость / сверка выплат)

## Сценарий: “Что ухудшилось относительно прошлого периода?”

1. Включить сравнение периодов в селекторе
2. На `/app/economics` посмотреть Δ по маржинальному вкладу и марже
3. На `/app/economics/sku/:sku` посмотреть блок сравнения и тренды

## Сценарий: “Где заморожены деньги на складе?”

1. Открыть `/app/economics/inventory`
2. Список “топ по замороженному капиталу” → выявить точки давления
3. Отдельно посмотреть slow movers и dead stock
4. При отсутствии себестоимости — импортировать себестоимость, чтобы оценка замороженного капитала стала точнее
